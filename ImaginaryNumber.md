# MH_허수(Imaginary Number)

- 실수 수직선
    1. 수직선 위의 점 1에 -1을 곱하면 -1이 됨
    2. 다시 -1을 곱하면 1이 됨
        
        ![image.png](image.png)
        
    3. 기하학적 입장에서 “-1을 곱하는 것은 원점을 중심으로 180도 회전 시키는 것과 같음
        1. 1 * (-1) = -1(오른쪽으로 1칸 지점에서 반시계 방향으로 180도 회전하여 왼쪽 1칸 지점으로 이동)
        2. -1 * (-1) = 1(다시 반시계 방향으로 180도 회전하여 원래 자리로 복귀)
        3. 즉, -1은 **[180도 회전 연산자]**라고 볼 수 있음
- 2차원으로의 확장(복소평면)
    - 확장이유
        - 180도 회전을 똑같은 연산으로 반으로 쪼개면 90도 회전
        - 90도 회전을 두 번하면 180도 회전
        - 하지만 1차원 실수 수직선 위에서는 90도 회전해서 갈 수 있는 공간 없음
    - 세로축 추가로 2차원 평면 확장 ⇒ **[복소평면(Complex Plane)]**
        - 가로축(Real Axis) : 기존의 실수 수직선
        - 세로축(Imaginary Axis) : 허수가 존재하는 수직선

- 2허수단위 i
    - 제곱했을 때 음수가 되는 수, 실수에는 존재하지 않음
        - $i = \sqrt{-1} \quad \text{또는} \quad i^2 = -1$
        - ex : $\sqrt{-4} = \sqrt{4} \times \sqrt{-1} = 2i$
        - 방정식의 해 완결 : $x^2+1=0$
        - 복소수 : $(a + bi)$
    - 기하학적 의미(90도 회전)
        - 어떤 수 * -1 ⇒ 수직선 위에 원점을 기준으로 180도 회전 시키는 것과 같음
            - 1 * (-1) = -1(180도 회전) ⇒ -1 * -1 ⇒ 1(180도 회전)
